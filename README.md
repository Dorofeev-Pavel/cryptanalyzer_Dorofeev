# Caesar Cipher Tool

Программа для шифрования и расшифровки текста с использованием шифра Цезаря. Поддерживает работу с большими файлами, валидацию входных данных, а также опциональные режимы brute force и статистического анализа.

## Возможности

- Шифрование и расшифровка текста с заданным ключом (сдвигом).
- Работа с большими текстовыми файлами (чтение/запись).
- Валидация входных данных (существование файлов, допустимость ключей).
- Режимы работы:
  - Шифрование текста.
  - Расшифровка с известным ключом.
  - Расшифровка методом brute force (перебор всех ключей).
  - Расшифровка методом статистического анализа (автоматический подбор ключа).
- Интерфейс: текстовое меню или опционально графический интерфейс.
- Обработка ошибок и исключений.
- Оптимизация для производительности.
